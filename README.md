# Portfolio-Sara_website

Portfolio-Sara_website is a simple personal portfolio website developed using **HTML** and **basic CSS**. The website includes a **Home Page** and a **Contact Page** with a clean and user-friendly design.

This project was created to practice and improve front-end web development skills, focusing on webpage structure, styling, layout design, and basic navigation using HTML and CSS.

## Features
- Home Page
- Contact Page
- Simple and clean user interface
- Basic CSS styling
- Beginner-friendly project

## Technologies Used
- HTML
- CSS
